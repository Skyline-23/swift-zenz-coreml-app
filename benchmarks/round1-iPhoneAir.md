## 所見 / 소견 / Observations

- **日本語:** iPhone Air での結果は **Stateless Greedy / Sync main** が最も高速で、短文では 0.03–0.7s 台、長文では 0.6–1.6s 台を示しました。.<br>ただし `Stateless Greedy / Async global` に大きな外れ値（6s, 18s, 7.7s 等）があり、平均を押し上げています。これらは Core ML スレッドの遅延や ANE コンテキストのスケジューラ問題が疑われます。

- **한국어:** iPhone Air에서는 **Stateless Greedy / Sync main** 구성이 전체적으로 가장 빠릅니다.<br>짧은 입력은 대체로 0.03~0.7초, 긴 입력은 0.6~1.6초 범위였고, `Stateless Greedy / Async global`에 6s, 18s, 7.7s 같은 큰 이상치가 포함되어 평균이 크게 올라간 것으로 보입니다. Core ML 스레드 동기화 지연 또는 ANE 스케줄러 이슈를 의심해야 합니다.

- **English:** On iPhone Air the **Stateless Greedy / Sync main** configuration is the fastest overall. Short prompts typically run in ~0.03–0.7s, long prompts around ~0.6–1.6s..<br>Note that the **Stateless Greedy / Async global** column contains large outliers (6s, 18s, 7.7s) that inflate its mean — likely due to Core ML thread sync latency or ANE context scheduler issues.
